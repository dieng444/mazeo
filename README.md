# Mazeo Framework

Mazeo is php5 framework that allow developer to build strong and secure php web app

##Depencies

- Twig template
- PHP unit
- PHP Password Agent

##Features

- Routing system with yaml files
- Form Manager
- ORM (Based on postgres sql)
- Entity Manager
- Template rendering with twig
- Authentication
- Reseting password
- Role management
- Session Management
- FlashInfo
- Locale (Translator)
